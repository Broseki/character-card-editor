# Character Card Editor

A web-based editor for creating and editing character cards used in AI roleplay applications. Supports the Tavern Card specification versions V1, V2, and V3.

**Try it online:** [ccedit.fur.is](https://ccedit.fur.is)

## Features

- **Multi-version support** - Edit cards in V1, V2, or V3 format with easy version switching
- **PNG import/export** - Load character data from PNG files and embed edited data back into images
- **JSON import/export** - Import and export raw JSON card data
- **Character Book editor** - Create and manage lorebook entries (V2+)
- **Asset management** - Manage character assets like icons and backgrounds (V3)
- **Auto-save** - Automatically saves your work to localStorage
- **Save slots** - Save multiple cards locally for easy access
